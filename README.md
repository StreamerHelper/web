# StreamerHelper Web

StreamerHelper Web Dashboard - monitoring, recording, and auto-uploading live streams.

## Features

- Multi-platform support (Bilibili, Huya, Douyu)
- Auto stream recording
- Auto upload to Bilibili
- Streamer management
- Content browsing

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- TanStack Query

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
